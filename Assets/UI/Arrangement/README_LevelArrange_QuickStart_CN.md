# Level Arrange 超短版

## 1 分钟流程

1. 左侧章节列表选中一个章节（`+ 大关卡` 可新增）。
2. 底部“未分配关卡”点 `+`，把关卡加入当前章节。
3. 中间关卡列表直接拖拽可改顺序（序号自动更新）。
4. 右侧看关卡预览（尺寸、标签、评分、可通关、备注）。
5. 顶部点 `保存` 写入 `campaign.json`。

## 高频操作

- **章节**
  - 单击选中；
  - 双击章节名可重命名；
  - 右侧预览区可编辑章节信息（名称、备注、解锁条件、是否上线）。
- **关卡**
  - 中间列表点 `x`：从当前章节移除（不会删文件）；
  - 底部 `新建关卡`：创建空关卡文件并加入可分配列表；
  - 右侧 `编辑`：自动保存 campaign 后跳转到 Level Editor 编辑该关卡。

## 退出

- `返回` 会弹确认：
  - `保存并退出`：先保存 campaign 再回主场景；
  - `不保存退出`：直接返回。

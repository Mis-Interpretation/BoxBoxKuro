# Level Editor 超短版

## 1 分钟流程

1. 左侧选 `Brush`，选子模式（点/线/面/边缘）和实体类型。
2. 鼠标左键放置、右键擦除；`Ctrl+Z` 撤销。
3. 鼠标中键滚轮缩放画布，按住拖拽画布。
4. 要移动/批量删时切到 `Select`：
   - 拖拽框选；
   - 拖动已选对象可整体移动；
   - `Shift` 多选，`Delete` 删除选中。
5. 顶部点 `Save` 保存关卡。
6. 点 `Validate` 试玩验证，`Esc` 退出试玩；点 `Solve` 自动求解。

## 必记

- `Brush` 负责“画”，`Select` 负责“改”。
- 网格尺寸在顶部改，范围 `1~50`。
- `Center` 可把当前实体整体居中。
- `Metadata` 面板可编辑标签/评分/备注/BGM 等元信息。
- `Save & Close`（Metadata）只写元数据；关卡布局仍用顶部 `Save`。

## 常见坑

- 鼠标压在 UI 上时，画布不响应是正常行为。
- 移动失败通常是越界或碰撞冲突（尤其文本占格冲突）。
- Metadata-only 保存前，最好先做一次普通 `Save` 创建关卡文件。

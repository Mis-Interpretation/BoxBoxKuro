给LevelEditor加一个新功能：选择模式

用来决定鼠标行为的效果

目前的点，线，面，框属于“笔刷模式”这个大模式。选择模式是一个和笔刷模式齐平的新大模式

选择模式下，（就像笔刷模式一样）用户可以选择点，线，面，框模式的选择

## 鼠标行为设计：

1. 当鼠标从一个没有已选择的物体区域开始长按时（并拖拽时），代表选择这些物体。根据模式不同，选择的范围不同；当在已被选择的物体区域开始长按（并拖拽时），代表拖动这些物体。拖动物体到新区域视为“从原本的地方删除这些物体”和“在新地方放置这些物体”的逻辑。能否overlap和之前的笔刷模式相同
2. 如果是单点模式，那么长按自动视为拖拽
3. 选择模式下，按住shift视为多选
4. 选择模式下，按Delete视为删除选中的内容

## UI上
1. 和笔刷模式一样在左边栏，占一格，点击UI会弹出flyerwindow选择模式
2. 为了和笔刷模式区分，这两个模式的格子下方都应当标注这是笔刷模式还是选择模式。
3. 记得和GridView配合哦，选择的部分应当也有一定程度的视觉效果作为指示。
